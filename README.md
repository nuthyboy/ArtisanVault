# ArtisanVault

A creative arts and craftsmanship rewards platform built on Stacks blockchain using Clarity smart contracts.

## Overview

ArtisanVault enables artisans and creators to showcase their craft mastery, earn artistry rewards, and participate in a decentralized creative guild ecosystem. The platform celebrates craftsmanship through transparent skill recognition and rewards.

## Features

- Craft discipline recognition and tracking
- Craft point accumulation system
- Artistry-based reward distribution
- Guild master management
- Transparent mastery progression

## Smart Contract Functions

### Public Functions
- `establish-artisan-guild` - Initialize creative guild with guild master
- `recognize-craft-discipline` - Recognize craft disciplines for tracking
- `record-craft-progress` - Record artisan craft development
- `host-artistry-showcase` - Calculate and distribute artistry rewards
- `complete-master-artisan-certification` - Complete certification and claim rewards

### Read-Only Functions
- `get-artisan-mastery` - Get artisan's craft mastery level
- `get-craft-discipline` - Get artisan's discipline specialization
- `get-total-craft-points` - Get total platform craft points
- `is-craft-recognized` - Check if craft discipline is recognized

## Getting Started

1. Deploy the contract to Stacks blockchain
2. Establish artisan guild with guild master
3. Recognize craft disciplines for tracking
4. Begin recording craft progress
5. Host showcases and claim artistry rewards
